##### Utilizar JAVA 16

- Para rodar localmente o docker-compose:
```
$ mvn package
$ docker-compose up
```

- Acessar Swagger no endereço:

```
http://localhost:8093/swagger-ui.html#/
```

-----

- #### Rodando via spring

```
mvn spring-boot:run
```

- Acessar Swagger em ambiente de DEV na porta 8092

```
http://localhost:8092/swagger-ui.html#/
```
